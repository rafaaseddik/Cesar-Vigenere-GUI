# Cesar/Vigenere GUI
#### A GUI to crypt/decrypt and crypto-analyze in Cesar and Vigenere Cyphers

### How to run 
To run this app, execute the "App GUI.py" file on the terminal : 
```
$ python "App GUI.py"
```

### Dependencies 
 * Pandas
 * PILLOW
